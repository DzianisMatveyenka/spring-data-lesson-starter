Создать SpringData репозитории для работы с сущностями Director и Movie.
Написать следующие методы и тесты для них:

- Поиск фильма по имени
- Поиск режиссёра по имени и фамилии
- Поиск списка фильмов по жанру
- Поиск списка фильмов по списку жанров
- Поиск фильма по имени и фамилии режиссёра
- Поиск списка фильмов между заданными датами
- Поиск фильма по частичному совпадению названия и без учёта регистра
- Поиск списка режиссёров по жанру фильма
